# 🚀 .NET Microservices Architecture

A practical study project focused on building modern microservices using ASP.NET Core, Clean Architecture, CQRS, MediatR, RabbitMQ, MassTransit, Docker, SQL Server, PostgreSQL, Redis and Event-Driven Architecture.

> ⚠️ This repository is continuously evolving as I progress through my studies and implement new architectural patterns and technologies.

Sobre o projeto

Este projeto faz parte da minha jornada de estudos em arquitetura de microsserviços com .NET. O objetivo é colocar em prática padrões, conceitos e tecnologias amplamente utilizados no desenvolvimento de aplicações distribuídas, simulando um ambiente próximo ao encontrado em projetos reais.

Durante o desenvolvimento, estou implementando diferentes microsserviços, explorando desde a arquitetura da aplicação até a comunicação assíncrona entre serviços e a containerização de toda a infraestrutura.
